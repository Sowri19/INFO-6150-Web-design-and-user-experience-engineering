# To-Do API

APIs available in this service


```
//
Retrieve all contacts
GET /TODO Item

//Create new TODO item
POST /TODO Item

//Retrieve a single TODO by id.
GET /TODO Item/:id

//Update a single TODO by id
PUT /TODO Item/:id

//Remove a single TODO by id
DELETE /TODO Item/:id

```

Tools used

1.`expressjs`
2.`mongoosejs`
